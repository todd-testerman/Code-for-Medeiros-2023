# Data Directory

This directory should contain the QIIME2 output files required for the analysis.

## Required Files

| File | Description |
|------|-------------|
| `table.qza` | Feature table containing ASV/OTU counts |
| `rooted-tree.qza` | Phylogenetic tree for diversity calculations |
| `taxonomy.qza` | Taxonomic classifications for each feature |
| `Alzheimers_metadata.txt` | Sample metadata with cohort assignments (Baseline, Control, Probiotic) |

## Data Source

These files are generated from QIIME2 processing of 16S rRNA sequencing data from Alzheimer's disease patient samples.

## Note

Data files are not included in this repository due to size and privacy considerations. Contact the corresponding author for data access.

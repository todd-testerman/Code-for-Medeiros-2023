# Figures Directory

Output figures from the analysis are saved here.

## Generated Figures

| File | Description |
|------|-------------|
| `phylum_comp.png` | Box plot comparing Bacteroidetes and Firmicutes relative abundance across cohorts |

## Figure Details

The main figure shows:
- Relative abundance of Bacteroidetes and Firmicutes phyla
- Three cohorts: Baseline, ADC (Control), and ADP (Probiotic)
- Statistical comparisons using t-tests with Bonferroni correction
- Significance annotations (* p < 0.05, ns = not significant)
